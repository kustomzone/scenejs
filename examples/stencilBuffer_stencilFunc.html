SceneJS - setting the stencil buffer comparison function

In this demo we use stencil Buffer

The first stage we draw the box and write the stencil value to 9 if there's a fragment. We then draw the teapot in the second stage and preserve the fragments based on the stencil value.
We intervally swtich stencil func of the second stage (teapot) between 'less', 'greater', 'never' and 'always'
